List
ОСНОВНАЯ ФУНКЦИОНАЛЬНОСТЬ:
описывает упорядоченную коллекцию, в которой имеет значение расположение элемента,
сохраняет последовательность добавления элементов,
позволяет осуществлять доступ к элементу по индексу.
ПРИМЕРЫ ТИПИЧНОГО ИСПОЛЬЗОВАНИЯ:
динамически изменяющиеся списки,
при необходимости быстрого доступа к элементу по индексу

Set
ОСНОВНАЯ ФУНКЦИОНАЛЬНОСТЬ:
содержит уникальные элементы,
позволяет быстро находить существующий элемент.
ПРИМЕРЫ ТИПИЧНОГО ИСПОЛЬЗОВАНИЯ:

Queue
ОСНОВНАЯ ФУНКЦИОНАЛЬНОСТЬ:
предназначена для хранения элементов в порядке, нужном для их обработки.
ПРИМЕРЫ ТИПИЧНОГО ИСПОЛЬЗОВАНИЯ:
очереди

Map
ОСНОВНАЯ ФУНКЦИОНАЛЬНОСТЬ:
содержит ключи и значения,
не может содержать дубли ключей.
ПРИМЕРЫ ТИПИЧНОГО ИСПОЛЬЗОВАНИЯ:
словари, properties-файлы
